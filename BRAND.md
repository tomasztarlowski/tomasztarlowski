# ivvo Signal Standard

This repo now carries a simple brand system for the GitHub profile.

## Assets

- `assets/ivvo-mark.svg` — square monogram for avatars, favicons, social cards, stickers, and repo badges.
- `assets/ivvo-standard.svg` — wide banner for the GitHub profile README.
- `assets/profile-banner.svg` — previous profile banner kept as fallback/reference.

## Concept

The mark is built from four product signals hidden inside `IVVO`:

- `I` — Idea: the vertical starting point.
- `V` — Vision: the first directional decision.
- `V` — Value: the second decision, mirrored into delivery.
- `O` — Output: the shipped, visible result.

It is intentionally geometric, readable at small sizes, and not tied to a trend like generic gradients, mascot marks, or stock-code symbols.

## Usage

Use the mark when the surface is small. Use the standard when the surface needs to feel like a banner, cover, or manifesto.

Recommended background: `#0B1017`.

Primary colors:

- Signal teal: `#5EEAD4`
- Deep teal: `#0F766E`
- Signal blue: `#2563EB`
- Ember: `#F97316`
- Paper: `#ECFEFF`
